## Description

Inventory Managment to manage products

## Tecnologies

- TypeScript
- React
- Next.js
- Postgres
- Chakra UI
- Prisma

## Requirements

- Node.js ^20.13.1
- Yarn

## Installation

1. Install dependencies:

```bash
$ yarn install
```

2. Run application:

```bash
# development watch mode
$ yarn dev

# production mode
$ yarn build
$ yarn start
```
